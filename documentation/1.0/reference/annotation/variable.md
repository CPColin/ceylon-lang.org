# #{page.title}

The `variable` annotation marks a value that may be [assigned](../../operator/assign)
(as opposed to [specified](../../statement/specification).

## Usage

<!-- try: -->
    void example() {
        variable Integer i;
        i = 1;
        i = 2;
    }
